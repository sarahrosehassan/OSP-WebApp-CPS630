# 🛒 OSP Web Application – CPS630
This is a **web application project** for CPS630, focused on building an **Online Service Platform (OSP)** that provides:
- **Shopping Services** – Finding and selecting the best deals.
- **Delivery Services** – Finding the optimal delivery path.
- **Payment Services** – Secure payment processing.

The project is developed in **two iterations**:
- **Iteration I**: Client-side development using **HTML5, CSS3, JavaScript, and jQuery**.
- **Iteration II**: Backend integration using **PHP and MySQL**.
